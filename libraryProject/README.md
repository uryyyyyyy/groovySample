# LibraryProject

## setup

`./gradlew [idea | eclipse]`

you can open [idea | eclipse] project

## run dynamically

`./gradlew run`

## compile as simple jar

`./gradlew build`

you can get jar file `build/libs/libraryProject.jar`

## compile as fat-jar

`./gradlew fatJar`

you can get jar file in `build/libs/libraryProject-all.jar`
