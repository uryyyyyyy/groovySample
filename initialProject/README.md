# InitialProject

## setup

`./gradlew [idea | eclipse]`

you can open [idea | eclipse] project

## run dynamically

`./gradlew run`

## compile as executable jar

`./gradlew distZip`

you can get zip file in `build/distributions/`

## run executable jar

```
cd build/distributions/
unzip initialProject.zip
./bin/initialProject
```
